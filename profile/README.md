# Meworla

Meworla develops tools and frameworks for building interactive software and games.

## Main Projects

### Rhea Engine
A modular C++ game engine framework designed for flexibility and composability.

Repository:
https://github.com/rhea-engine/rhea

Documentation:
https://rhea.dev

### mwpkg
A package manager for engine modules and development tooling.

Repository:
https://github.com/rhea-engine/mwpkg

## Philosophy

Our projects focus on:

- modular architecture
- explicit systems
- developer-friendly tooling
- modern C++ foundations
